Count the number of pairs

Given an array of numbers. Count how many pairs of elements in it are equal to each other.
It is believed that any two elements that are equal to each other form one pair, which must be calculated.


  • The array "seen" is used to find duplicate numbers in a list of elements.
  • I used the formula n(nâˆ’1)/2 for the number of pairs from an n element.
For example:
Sample Input:
5
1 1 1 1 1
Sample Output:
10

C++Code 
																
#include  <iostream>
#include  <vector>
using namespace std;
int main( void ) {
 int N;
 cin >> N;//Count of numbers
    
int A[N]={1, 2, 3, 2 ,3};
vector  b(N);
for(int i = 0; i < N; i++)
cin >> A[i];
int seen[N];
for(int i = 0; i < N; i++)
seen[i] = 0;
for(int i = 0; i < N; i++) {
if(seen[i] == 0) {
int count = 0;
for(int j = i; j < N; j++)
if(A[j] == A[i]) {
count += 1;
seen[j] = 1;
}
b[i]=count;
}
}
int s=0;
 for (int i = 0; i < N; i++) {
        if (b[i]>1) {
            s=s+(b[i]*(b[i]-1))/2;
        }
    }
    cout  << s;
return 0;
}
